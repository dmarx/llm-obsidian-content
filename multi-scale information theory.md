# Multi-scale Information Theory

[[Multi-scale information theory]] provides a unified framework for analyzing information structures and flows across multiple scales of organization in complex systems.

## Mathematical Foundations

### Scale Space

Define a scale space $(\Lambda, \preceq)$ where:
- $\Lambda$ is the set of scales
- $\preceq$ is a partial order on scales
- For $\lambda_1, \lambda_2 \in \Lambda$, $\lambda_1 \preceq \lambda_2$ indicates $\lambda_1$ is a finer scale than $\lambda_2$

### Information Measures

For each scale $\lambda$, define:

$I_\lambda(X) = -\sum_{x \in X_\lambda} p_\lambda(x) \log p_\lambda(x)$

where:
- $X_\lambda$ is the state space at scale $\lambda$
- $p_\lambda$ is the probability distribution at scale $\lambda$

## Cross-Scale Information Relations

### Scale Transfer Operator

The [[Scale Transfer Operator]] $T_{\lambda_1 \to \lambda_2}$ maps information between scales:

$T_{\lambda_1 \to \lambda_2}: I_{\lambda_1} \to I_{\lambda_2}$

satisfying:

$I_{\lambda_2} = T_{\lambda_1 \to \lambda_2}[I_{\lambda_1}] + \Delta I_{\lambda_1,\lambda_2}$

where $\Delta I_{\lambda_1,\lambda_2}$ represents information gained/lost in scale transition.

### Information Flow Network

Define the [[Multi-scale Information Network]] $G = (V,E)$ where:
- Vertices $V = \{\lambda_i\}$ represent scales
- Edges $E = \{(\lambda_i, \lambda_j, J_{ij})\}$ represent information flows
- $J_{ij}$ quantifies information flow between scales $i$ and $j$

## Theoretical Properties

### Scale Decomposition Theorem

For any system $S$:

$I_{total}(S) = \sum_{\lambda \in \Lambda} I_\lambda(S) - \sum_{\lambda_1,\lambda_2} I_{overlap}(\lambda_1,\lambda_2)$

where $I_{overlap}$ accounts for redundant information between scales.

### Information Conservation

At each scale transition:

$\Delta I_{\lambda \to \lambda'} + \Delta I_{\lambda' \to \lambda} \leq 0$

representing the [[Second Law of Information Thermodynamics]].

## Applications

### Dynamic Multi-scale Systems

For time-evolving systems:

$\frac{\partial I_\lambda}{\partial t} = D_\lambda \nabla^2_\lambda I_\lambda + F_\lambda(I_\lambda) + \sum_{\mu \neq \lambda} G_{\lambda\mu}(I_\lambda, I_\mu)$

where:
- $D_\lambda$ is scale-dependent diffusion
- $F_\lambda$ represents within-scale dynamics
- $G_{\lambda\mu}$ captures cross-scale interactions

### Pattern Recognition

[[Multi-scale pattern recognition]] utilizes scale-dependent features:

$P(\text{pattern}|\{I_\lambda\}) = \prod_{\lambda} P_\lambda(\text{pattern}|I_\lambda)$
